## Overview

**AicuSa Interiors** is a fully responsive, multi-page front-end website for a premium interior design firm. Built with HTML5, CSS3, Bootstrap 4, and vanilla JavaScript — no build tools, no frameworks, just clean and deployable code straight from the browser.

The site presents the firm's services, portfolio, team, testimonials, and blog in a polished, professional layout that works flawlessly across all screen sizes.

## Pages

|------|-------------|
| `index.html` | Home — hero carousel, about, services, portfolio, team, testimonials, blog |
| `about.html` | Company story, values, and team |
| `service.html` | Full services catalogue |

| `contact.html` | Contact form with embedded map |

## Features

| Feature | Description |
|---------|-------------|
| Hero Carousel | Auto-sliding fullscreen banner with text overlay and CTA buttons |
| Isotope Portfolio Filter | Client-side grid filtering by All / Complete / Running / Upcoming |
| Owl Carousel | Smooth services and testimonials slider |
| Lightbox Gallery | Click-to-enlarge image viewer for portfolio items |
| Contact Form | Bootstrap-validated form with PHP email backend |
| Smooth Scroll | Eased anchor-link navigation throughout |
| Back to Top | Animated scroll-to-top button on all pages |
| Fully Responsive | Tested from 320px mobile to wide desktop |
| Icon Libraries | Font Awesome 5 and Flaticon integration |

## Getting Started

No build step or package manager required.

**1. Clone the repository**

```bash
git clone https://github.com/satviklandge/Interior_Design_website.git
cd Interior_Design_website
```

**2. Open in browser**

```bash
# Quickest — just open the file
open index.html

# Recommended — use VS Code Live Server for hot reload
# Right-click index.html -> Open with Live Server

# Or use Python's built-in server
python -m http.server 8000
# Visit http://localhost:8000
```

> The contact form requires a PHP-capable server to send emails. For local development, tools like [XAMPP](https://www.apachefriends.org/) or [MAMP](https://www.mamp.info/) will handle this.

## Responsive Breakpoints

```
< 576px     Mobile    — single column, hamburger nav
576-768px   Small     — 2-column grid begins
768-992px   Tablet    — full navigation bar visible
992-1200px  Desktop   — multi-column layout active
> 1200px    Large     — max-width container centered
```
